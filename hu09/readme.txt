Gestión de Tareas con Spring Boot
Este es un programa de gestión de tareas desarrollado con Spring Boot. El programa te permite crear, leer, actualizar y eliminar tareas. Además, también te permite reprogramar una tarea existente.


Funcionalidades
Crear una nueva tarea: puedes agregar una nueva tarea a la lista de tareas pendientes. Cada tarea debe tener un nombre y una fecha límite.
Leer tareas existentes: puedes ver una lista de todas las tareas pendientes.
Actualizar tareas existentes: puedes cambiar el nombre y/o la fecha límite de una tarea existente.
Eliminar tareas existentes: puedes eliminar una tarea existente de la lista de tareas pendientes.
Reprogramar tareas existentes: puedes cambiar la fecha límite de una tarea existente.


Tecnologías utilizadas
Este programa se ha desarrollado utilizando las siguientes tecnologías:

Java 11: lenguaje de programación utilizado en la implementación del programa.
Spring Boot: marco de trabajo utilizado para crear aplicaciones Java rápidamente.
Thymeleaf: motor de plantillas utilizado para generar la interfaz de usuario del programa.
HTML, CSS y JavaScript: lenguajes utilizados para diseñar y dar estilo a la interfaz de usuario del programa.


Instalación
Para ejecutar este programa en tu ordenador, sigue estos pasos:

Descarga o clona el repositorio desde GitHub.
Abre el proyecto en tu IDE (IntelliJ IDEA, Eclipse, NetBeans, etc.).
Espera a que se descarguen todas las dependencias de Maven.
Ejecuta la clase principal TareaApplication.
Abre un navegador web y visita la dirección http://localhost:8080/tarea.
